# ch03 실습과제1

## 문제1
단위행렬(identity), 역행렬(inverse), 전치행렬(transpose)의 용어를 설명하라. 

## 답
단위행렬(identity) : 행렬의 대각선의 성분이 모두 1이고 나머지 성분은 0인 행렬 <br>
역행렬(inverse) : 어떤 행렬과 곱셈을 하였을 때 결과가 단위행렬인 행렬 -> A*A의 역행렬=I(단위행렬) <br>
전치행렬(transpose) : 행과 열을 서로 바꾼 행렬 <br>

## 문제2
파이썬 리스트와 넘파이 ndarray의 데이터 저장방식을 비교하고 넘파이가 처리속도가 빠른 이유를 설명하라.

## 답
Python 리스트는 각 원소가 따로 저장되는 참조형 구조라 접근이 느리고, 동적 타이핑 때문에 매번 타입을 확인해야 한다. <br>
하지만 NumPy 배열은 같은 타입 데이터를 연속된 메모리에 저장하는 정적 구조라 캐시 효율이 좋고, 타입 검사 없이 바로 연산이 가능하다. <br>
또한 NumPy는 C로 구현되어 있고 SIMD 같은 벡터 연산을 사용해 여러 데이터를 한 번에 처리하므로 Python보다 훨씬 빠르다.<br>
