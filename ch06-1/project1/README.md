# ch06-1 실습과제1

## 문제1
예제에서 보듯이 반복문을 이용한 화소처리와 opencv함수를 이용한 처리의 연산시간의 차이가 크게 나는 이유를 설명하라.<br>

## 답
반복문을 이용한 화소 처리는 각 픽셀을 하나씩 직접 접근하여 연산하므로 반복 횟수가 많아질수록 처리 시간이 크게 증가한다. <br>
반면 OpenCV 함수는 내부적으로 C/C++ 수준에서 최적화되어 있으며, SIMD 명령어, 멀티스레딩, 메모리 접근 최적화 등 다양한 성능 향상 기법을 사용하여 여러 화소를 효율적으로 처리한다. <br>
따라서 동일한 작업이라도 OpenCV 함수를 이용한 처리가 반복문으로 직접 구현한 방법보다 연산 시간이 훨씬 짧게 나타난다.<br>
